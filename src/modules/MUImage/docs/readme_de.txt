INSTALLATIONSANLEITUNG
----------------------
1) Kopieren Sie MUImage in das Verzeichnis modules.
2) Initialisieren und aktivieren Sie MUImage in der Modulverwaltung.
3) Setzen Sie das Verzeichnis `userdata/MUImage/` inklusive aller Unterordner auf beschreibbar.
