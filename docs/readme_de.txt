INSTALLATIONSANLEITUNG
----------------------
1) Kopieren Sie MUImage in das Verzeichnis modules.
2) Laden Sie den Ordner MUImage in Resources/userdata in den Ordner userdata im root Verzeichnis von Zikula
3) Setzen Sie das Verzeichnis `userdata/MUImage/` inklusive aller Unterordner auf beschreibbar.
4) Initialisieren und aktivieren Sie MUImage in der Modulverwaltung.
