INSTALLATION INSTRUCTIONS
-------------------------
1) Copy MUImage to your modules folder.
2) Put the folder MUImage in Resources/userdata into the folder userdata in the root directory of Zikula
3) Make the directory `userdata/MUImage/` writable including all sub folders.
4) Initialize and activate MUImage in the modules administration.
